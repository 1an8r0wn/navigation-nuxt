# Navigation Nuxt

基于 Nuxt 3 框架构建的开源个人导航平台。

## 安装

```bash
pnpm install
```

## 运行

项目运行时可在 `http://localhost:3000` 查看

```bash
pnpm run dev
```

## 部署到 Docker

```bash
docker-compose up -d
```

## 发布

```bash
pnpm run build
```

本地预览

```bash
pnpm run preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
