# Navigation Nuxt

基于 Nuxt 3 框架构建的开源个人导航平台。

## 安装


```bash
# pnpm
pnpm install
```

## 运行

项目运行时可在 `http://localhost:3000` 查看

```bash
# pnpm
pnpm run dev
```

## 发布

```bash
# pnpm
pnpm run build
```

本地预览

```bash
# pnpm
pnpm run preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
